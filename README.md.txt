Tomato is a food delivery web app that connects users with their favorite local restaurants. Fast, fresh, and easy to use.

## Features
- Browse and search restaurants
- Add items to cart
- Real-time order tracking
- Secure checkout

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB

## Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/tomato.git

# Install dependencies
npm install

# Start the app
npm start
